# Proofreader AI:

This document outlines the full-stack architecture for a Proof of Concept (POC) using **React**, **FastAPI**, **MongoDB**, and **Gemini 2.5 Flash-Lite**.

---

## 1. Project Overview & Stack

* **Frontend:** React, Tailwind CSS (Styling), Axios (API calls).
* **Backend:** FastAPI (Python 3.10+), Pydantic (Validation).
* **Database:** MongoDB (via `motor` for async support).
* **AI:** Google Gemini 2.5 Flash-Lite Free tier (via `google-genai` SDK).
* **Auth:** JWT (JSON Web Tokens) with password hashing.

## 4. System Workflow

### User Lifecycle

1. **Signup:** User provides credentials $\rightarrow$ Backend hashes password $\rightarrow$ Entry created in MongoDB.
2. **Login:** Backend verifies hash $\rightarrow$ Returns JWT $\rightarrow$ Frontend saves JWT in `localStorage`.
3. **Auth Check:** On page refresh, Frontend calls `/auth/me`. If 401, redirect to `/login`.

### Analysis Lifecycle

1. **Submission:** User pastes text in React $\rightarrow$ Clicks **Analyze**.
2. **UI Lock:** Button disabled $\rightarrow$ Spinner appears $\rightarrow$ `beforeunload` listener active.
3. **API Call:** FastAPI receives text + User ID $\rightarrow$ Calls Gemini with `response_schema`.
4. **Database:** Gemini JSON result saved to `history` collection with timestamp.
5. **Completion:** React receives JSON $\rightarrow$ Displays result $\rightarrow$ Updates sidebar (Top 10).

## 6. Backend Function Definitions

### `ai_service.py` (Structured Gemini Call)

```python
async def get_proofread_analysis(text: str):
    # Forced JSON output using the AnalysisResult schema
    config = {
        "response_mime_type": "application/json",
        "response_schema": AnalysisResult,
    }
    response = client.models.generate_content(
        model="gemini-2.5-flash-lite",
        contents=f"Analyze this text: {text}",
        config=config
    )
    return response.parsed # Returns a validated Python object

```

### `auth_utils.py` (Security)

* `get_password_hash(password)`: Hashes plain text.
* `verify_password(plain, hashed)`: Boolean check for login.
* `create_token(user_id)`: Generates JWT with 1-hour expiry.

## 7. Frontend UX Logic

### The "Safety Guard" (React)

To prevent users from losing their results during the processing phase:

```javascript
useEffect(() => {
    const lockPage = (e) => {
        if (isLoading) {
            e.preventDefault();
            e.returnValue = "Analysis in progress. Are you sure you want to leave?";
        }
    };
    window.addEventListener("beforeunload", lockPage);
    return () => window.removeEventListener("beforeunload", lockPage);
}, [isLoading]);

```

### Sidebar History

The sidebar should perform a `GET /history` request.

* **Query:** `find({"user_id": uid}).sort("timestamp", -1).limit(10)`
* **UI:** Clicking a history item populates the main result box without re-calling the AI.
